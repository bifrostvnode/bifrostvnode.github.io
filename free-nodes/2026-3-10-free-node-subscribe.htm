  1. 打开V2RayNG，点击底部导航栏的"订阅"选项。

  1. 返回主界面，您将看到所有可用的节点列表。

常见问题解决方案

  1. 检查订阅是否过期，尝试更新订阅。
  2. 切换不同协议类型（如从VMess改为VLESS）。
  3. 尝试不同的传输方式（如TCP改为WebSocket）。
  4. 检查本地网络是否正常，尝试切换WiFi/移动数据。

速度慢如何优化？

  1. 选择地理位置较近的节点（如日本、新加坡节点对中国用户通常较快）。
  2. 关闭不必要的加密选项以降低CPU开销。
  3. 在夜间等非高峰时段使用，避开网络拥堵。
  4. 考虑使用付费节点获取更好的带宽保障。

节点频繁掉线怎么处理？

  1. 启用"自动切换节点"功能，设置合理的切换阈值。
  2. 在"设置"中调整心跳间隔和超时时间。
  3. 使用支持TCP BBR加速的节点改善长距离连接稳定性。

FAQ

V2Ray 策略组如何根据节点延迟自动选择？
策略组会根据节点延迟和丢包率自动选择最优节点。结合路由规则可保证国内流量直连，海外流量走最佳节点，提高访问速度和连接稳定性，优化整体网络体验。
V2Ray 如何通过策略组优化访问速度？
策略组允许用户将节点按延迟、地域或用途分组。结合路由规则，V2Ray 会自动选择最优节点处理流量，提升访问速度并降低延迟，尤其在访问海外网站时效果明显。
Trojan 的伪装域名有什么要求？
Trojan 的伪装域名必须是可以通过 HTTPS 正常访问的真实网站，并且 SSL 证书有效。伪装域名用于与服务器证书匹配，使得流量在外观上完全与正常 HTTPS 请求一致。
VLess + Reality 的优势主要体现在哪些方面？
Reality 采用非传统 TLS 流量伪装方式，通过模拟目标网站证书与握手特征来隐藏代理特征，相较于普通 VLess + TLS 更难被识别。它还具备更高的性能与更少的明文特征，适合高安全需求环境。
Sing-box 如何按域名分流优化浏览器流量？
Sing-box 可通过规则匹配域名，将浏览器流量分配到最优节点。结合策略组，可保证网页访问速度和稳定性，同时优化视频和游戏等其他流量，提高整体网络体验。
Hysteria2 多路复用对高频连接的优势？
多路复用可让多个短连接共享同一通道，减少握手和加密开销，提高吞吐量。适合高频短连接场景，如即时通讯和游戏，保证流量快速、稳定传输。
如何将 Windows 上的 V2Ray 与 WSL 环境联合调试？
在 WSL 内运行需要的调试工具并通过 localhost 或将 V2Ray 监听地址绑定为 0.0.0.0，然后在 Windows 与 WSL 之间配置端口映射。注意 Windows 防火墙规则需允许 WSL 与 Windows 进程通信。
VLESS + XTLS 模式适合高并发环境吗？
适合。XTLS 去除冗余加密，提高 CPU 使用效率，降低延迟。结合策略组和多出口节点，可在高并发或多连接场景下保持稳定、高速的网络体验。
Surfboard 如何结合规则实现智能节点切换？
Surfboard 可根据规则文件和策略组自动选择延迟低、稳定性高的节点处理流量。不同应用流量可分配不同节点，优化访问速度和稳定性，减少手动切换节点的操作。
VMess AEAD 加密在安全性方面有何优势？
AEAD 加密可防止数据篡改、重放攻击和流量特征泄露。在高封锁环境下保证节点可用，提升数据安全和连接稳定性，同时降低被检测风险，确保长期安全访问。
