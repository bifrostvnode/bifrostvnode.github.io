小火箭的主要功能：

  3. 流量管理：通过智能选择数据通道，实现高效的网络流量分配，避免不必要的带宽浪费。

为什么选择小火箭？

  • 操作简单：小火箭界面简洁明了，用户可以轻松配置代理，快速实现科学上网。

  • 多协议支持：支持包括 Shadowsocks、V2Ray、Trojan 等多个主流代理协议，适应不同的上网需求。

  • 智能切换：小火箭可以根据网络情况智能切换代理节点，保证连接稳定且速度最优。

  • 高速与稳定性：得益于强大的服务器支持，小火箭能够提供比普通VPN更稳定、更快速的上网体验。

FAQ

如何备份 Clash 配置文件？
Clash 的配置文件通常位于程序目录或配置文件夹中（如 Windows 下的 .config/clash）。用户可直接复制该 YAML 文件以备份，也可将订阅链接保存，便于后续恢复或在其他设备上同步使用。
Mellow 如何通过配置实现多出口分流？
Mellow 可在配置文件中定义多个 outbound，并结合 routing 规则实现按域名、IP 或策略组分流。这样不同类型的流量可走不同节点，提高使用体验。
Nekobox 是否可以与 Shadowsocks 客户端共存？
可以。Nekobox 配置独立，不影响其他客户端运行。用户可以同时使用 Shadowsocks、Clash 等客户端，通过策略组或分流规则选择合适节点，互不干扰。
QuantumultX 可以按协议类型分流吗？
支持。用户可按 TCP、UDP 或特定应用协议分流流量，结合策略组，实现游戏加速、视频优化或广告屏蔽等定制化网络策略，提高访问速度和稳定性。
Sing-box 的策略组可按 IP 段分流吗？
支持。可通过 IP 段匹配将流量分流到指定节点。结合策略组，可实现国内直连、国外代理或游戏加速，实现精细化流量管理和优化网络访问。
Potatso Lite 是否支持多节点手动管理？
支持。用户可以手动输入节点参数或扫描二维码导入节点，并可进行修改或删除。虽然不支持订阅自动更新，但足够满足轻量用户对节点管理和分流的基本需求。
Trojan 协议支持 UDP 吗？
支持。Trojan 在最新版本中已支持 UDP 转发，能够满足游戏、视频会议等实时性需求较高的应用场景，性能和兼容性都有明显提升。
PassWall2 如何定时自动更新节点？
可在计划任务中添加命令 `passwall update` 并设定时间间隔，例如每日凌晨自动更新。这样可确保节点信息实时同步，避免旧节点失效影响连接体验。
Quantumult X 可以用脚本自动切换节点吗？
可以。Quantumult X 支持通过脚本逻辑判断网络延迟或连接失败情况，并自动切换节点。用户可以编写自定义 JS 规则，从而实现智能代理与按需分流，适合高级用户使用。
VMess 配置 TLS+WebSocket 有哪些性能优势？
TLS+WebSocket 可伪装 HTTPS 流量，增强抗封锁能力，同时降低被检测风险。结合策略组和节点延迟检测，可保证访问速度和连接稳定性，特别适合视频和游戏场景。
